### 8. UI/UX 개선 관찰

#### MI-6: 태스크바 레짐 인디케이터
- **위치**: 태스크바 중앙, 레이아웃 버튼 좌측
- **표시**: 색상 dot(🟢/🟡/🔴) + 텍스트(안정/변동/위기) + 지속시간
- **추가 정보**: 마우스 hover 시 "시장 레짐: 평온 — 안정적 투자 환경 (N시간 지속)"
- **평가**: 직관적, 항상 보이는 위치 ✅

#### MI-2: 시장 개장/폐장벨 + 일일 P&L 토스트
- **폐장 토스트**: 매일 18시 화면 우하단에 표시
- **형식**: "오늘의 수익률: +0.XX% / 수고하셨습니다!" (양수) 또는 "-0.XX% / 내일은 더 좋은 하루가 될 거예요." (음수)
- **색상**: 📈 녹색(양수) / 📉 빨간색(음수)
- **지속시간**: 5초 후 자동 소멸
- **이슈**: 8x 속도에서 토스트가 빠르게 쌓임 (MAX_TOASTS=3으로 제한됨)
- **평가**: 하루의 리듬감을 제공, 속도 높을 때 약간 스팸성 ⚠️

#### 최악의 날 배너
- **발생**: 1995.03.18 일일 변동 -4.40%
- **형식**: Level 2 배너 "최악의 날... -4.4%"
- **표시**: 화면 좌하단 📉 아이콘 + 빨간 텍스트

## 종합 평가

### 확인된 기능 (5/15)
1. **MI-6** 태스크바 레짐 인디케이터 — 완벽 작동
2. **MI-2** 일일 P&L 토스트 — 작동 (속도 높을 때 스팸 경향)
3. **QW-6** 협상 4개월 주기 — 정확히 작동
4. **QW-10** 이벤트 체인 가중치 — 이벤트 체인 발생 확인
5. **celebrateWorstDay** — 최악의 날 배너 정상 표시

### 미확인 기능 (10/15)
- 직원 미배치로 매매 파이프라인 비활성 → MI-4 거래 축하, QW-7 제안서 만료 미확인
- 자산 하락 중 → QW-8 마일스톤, MI-9 등급 상승 미확인
- 직원 Lv.1 → QW-3 스킬 해금, QW-4 XP 증가 미확인
- 번아웃/서킷브레이커 미발생 → MI-1 자동 일시정지, QW-2 번아웃 완화 미확인

### 재미 개선 체감
1. **일일 리듬감 향상**: 폐장벨 + P&L 토스트가 "하루"라는 단위를 느끼게 해줌
2. **레짐 가시성**: 태스크바 인디케이터로 시장 상태 즉시 파악 가능
3. **협상 주기 단축**: 4개월 만에 리듬 게임 등장 → 인터랙션 빈도 증가
4. **이벤트 체인 활성화**: 금리 사이클 체인이 다단계 서사를 제공
5. **최악의 날 배너**: 극적 순간 강조 효과 우수

### 개선 필요 사항
1. 고속(8x+)에서 P&L 토스트 스팸 → 속도 4x 이상 시 억제 고려
2. 직원 미배치 시 안내 UX 필요 (현재 "미배치" 표시만)
3. 자산이 순수 지출로만 감소 → 초보 플레이어에게 매매 유도 필요
